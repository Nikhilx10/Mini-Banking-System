Mini Banking System using Streamlit

## Overview

A simple banking application built with Python and Streamlit that allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money through a web interface.

## Features

* Check account balance
* Deposit money
* Withdraw money
* User-friendly web interface
* Session-based balance storage

## Technologies Used

* Python
* Streamlit

## Project Structure

```
Mini-Banking-System/
│
├── app.py
├── requirements.txt
└── README.md
```

## How to Use

1. Select an operation from the dropdown menu.
2. Check your current balance.
3. Deposit money into the account.
4. Withdraw money if sufficient balance is available.
